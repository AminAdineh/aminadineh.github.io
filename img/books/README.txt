Place your images here for the Books, Interests & Concepts page.

Supported formats: JPG, PNG
Recommended size: 800x600px or larger (will be automatically resized)

To add images to the books.html page:
1. Place your image files in this directory (img/books/)
2. Edit books.html and add items following the example pattern

Example code for adding a new item:
<figure class="item lbimage" data-groups='["category_all", "category_books"]'>
  <a class="lightbox" title="Your Title" href="./img/books/your-image.jpg">
    <div class="portfolio-item-img">
      <img src="./img/books/your-image.jpg" alt="Your Title" title="" />
    </div>
    <div class="project-desc">
      <div class="project-desc-text">
        <h4 class="name">Your Title</h4>
        <span class="category">Books</span>
      </div>
    </div>
  </a>
</figure>

Available categories:
- category_books
- category_interests
- category_concepts
